Welcome to Coin Game!

Rules:

1. Each player starts with 20 coins

2. Same rule: If both players spend the same number of coins, player 2 gains 1 coin

3. Off-by-one rule: If the players do not spend the same number of coins and the positive difference between the number of coins spent by the two players is 1, player 2 is awarded 1 coin.

4. Off-by-two rule: If the players do not spend the same number of coins and the positive difference between the number of coins spent by the two players is 2, player 1 is awarded 2 coins.
